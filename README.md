# Homebrew Formula for Grant

This repo contains the [Homebrew](https://brew.sh/) formula for [Grant](https://github.com/anchore/grant)

## Installation

```sh
brew tap anchore/grant
brew install grant
```

## Issues, Feature Requests, etc...

If you have any problems related to the installation of Grant via Homebrew, please [open an issue](https://github.com/anchore/homebrew-grant/issues/new)

Any issues or feature requests related to Grant itself should be filed in the [Grant repo](https://github.com/anchore/grant)
